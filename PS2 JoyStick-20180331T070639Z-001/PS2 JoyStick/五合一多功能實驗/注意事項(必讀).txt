注意事項

1.請參閱所有實驗章節資料夾裡的接線圖將各組件連線接好.

2.若馬達運轉方向與預期不同時，您可以：

  (a)修改驅動馬達程式碼：
  例如, 原本驅動馬達（右）順時針轉動的程式碼如下：

  digitalWrite(MotorR_I1,HIGH);
  digitalWrite(MotorR_I2,LOW);

  如果實際通電後發現馬達運轉方向相反，則修改程式碼如下：
  digitalWrite(MotorR_I1,LOW);
  digitalWrite(MotorR_I2,HIGH);


  (b)將馬達接到驅動板藍色端子台的紅黑線位置對調：
  例如, 原本預期驅動馬達（右）順時針轉動，但實際通電後發現馬達運轉方向相反時，將馬達驅動板藍色端子台 MOTORA 的 紅色與黑色 導線位置對調.

3.試車前請先檢查 循跡模組 是否定動作正常.
  步驟 : (a)循跡模組前無任何物體時, 循跡模組上的 LED 是否熄滅, 若沒有熄滅請轉動旋鈕調整感測靈敏度至 LED 剛好熄滅即可.
         (b)遮住循跡模組上的感測裝置, 循跡模組上的 LED 是否亮起, 若沒有亮起請檢查旋鈕是否將感測靈敏度調的太低.

4.(a)請將 第4節：紅外線遙控實驗 資料夾裡的 IRremote 函式庫，加入 Arduion IDE 的 Library.
     步驟：開啟 Arduino IDE > 點選 Sketch > Import Library... > Add Library... > 選擇 IRremote 資料夾 > 點選 開啟 按鈕

  (b)使用 IRremote_Test 程式解出遙控器的內碼.

5.請將 Arduino自走車套件：PS2遙控實驗 資料夾裡的 PS2X_Lib 函式庫，加入 Arduion IDE 的 Library.
     步驟：開啟 Arduino IDE > 點選 Sketch > Import Library... > Add Library... > 選擇 PS2X_Lib 資料夾 > 點選 開啟 按鈕

6.HC-06 使用 UNO 的 Serial Port，所以上傳程式到 UNO 時，請務必先拔掉 HC-06 的 TXD 與 RXD 接線，待程式上傳完成再接回!




操控說明
1.小車上電後為標準模式
2.標準模式下可使用 紅外線遙控器、藍牙和PS2控制器來遙控小車
3.紅外線遙控器 "上"、"下"、"左"、"右" 按鍵 可控制小車方向
4.紅外線遙控器 "*" 按鍵 可設定小車進入 循跡模式, 要離開循跡模式請按下 紅外線遙控器 "OK" 鍵
5.紅外線遙控器 "#" 按鍵 可設定小車進入 超音波避障模式, 要離開循超音波避障請按下 紅外線遙控器 "OK" 鍵
6.PS2控制器 "上"、"下"、"左"、"右" 按鍵 可控制小車方向
7.PS2控制器 左右搖桿 各別控制 左右馬達 運轉方向
8.PS2控制器 "方形" 按鍵 可設定小車進入 循跡模式, 要離開循跡模式請按下 PS2控制器 "X" 鍵
9.PS2控制器 "圓形" 按鍵 可設定小車進入 循跡模式, 要離開循跡模式請按下 PS2控制器 "X" 鍵
